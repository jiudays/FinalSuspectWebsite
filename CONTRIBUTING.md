# 贡献指南

### 贡献方式

- **问题反馈：通过 issue 提交 bug 或功能建议**
- **代码贡献：fork 项目并提交 Pull Request**
- **文档改进：完善文档相关内容**
- **翻译支持：协助完善或增加多语言版本**

### 开发环境准备
- **HTML代码编辑器（如Visual Studio Code）**

### 贡献流程
#### **添加新功能**
- **Fork 项目到个人 GitHub 账户**
- **编写代码并确保符合项目规范**
- **提交 Pull Request 并详细描述改动内容**

#### **添加新语言翻译**
 - **打开 [issue](https://github.com/NewLezaiYa/FinalSuspectWebsite/issues) 申请添加新语言**

### 注意事项
**所有贡献需遵守 AGPL-3.0 License 许可证规定**
**确保改动不会破坏现有功能的正常运行**
